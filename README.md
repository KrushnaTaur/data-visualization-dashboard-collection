# 🤖 AI Tools Collection

A curated collection of **AI-inspired web applications (frontend simulations)** built using **HTML, CSS, and JavaScript**.
Each project replicates the user interface and basic logic of modern AI tools such as chatbots, generators, analyzers, and assistants.

This repository is part of my **Full Stack Development practice series**, focusing on building real-world AI product interfaces.
